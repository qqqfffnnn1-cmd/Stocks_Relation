# TK_StockRelation - 股票关联度分析工具

快速分析两只股票之间的关联度，包括价格相关性和资金共振分析。

## 功能特点

- **价格关联分析**：计算两只股票的价格相关系数、同步率、领先滞后关系
- **资金共振分析**：基于龙虎榜数据，分析两只股票的资金流向相似度
- **综合评分系统**：0-130分评分体系（价格0-100分 + 资金加成0-30分）
- **模糊搜索**：支持股票名称和代码的模糊搜索，覆盖全部A股（5484只）
- **Web界面**：简洁易用的网页界面

## 快速开始

### 方法1：双击启动（推荐）

1. 双击 `启动服务.bat` 启动服务
2. 浏览器会自动打开 http://localhost:5000
3. 在搜索框输入股票代码或名称（支持模糊搜索）
4. 点击"分析"按钮查看关联度报告

### 方法2：命令行启动

```bash
cd TK_StockRelation/web
python app.py
```

然后访问 http://localhost:5000

## 评分体系

### 总分构成
- **价格得分**（0-100分）：
  - 涨跌幅相关性（0-40分）
  - 同步率（0-60分）
- **资金加成**（0-30分）：龙虎榜席位重叠度

### 等级划分
- **S级（90+分）**：极强关联 ⭐⭐⭐⭐⭐
- **A级（70-89分）**：强关联 ⭐⭐⭐⭐☆
- **B级（50-69分）**：中等关联 ⭐⭐⭐☆☆
- **C级（30-49分）**：弱关联 ⭐⭐☆☆☆
- **D级（<30分）**：几乎无关联 ⭐☆☆☆☆

## 分析指标说明

### 价格相关性
- **涨跌幅相关系数**：衡量两只股票涨跌幅的线性相关性（-1到1）
- **同步率**：两只股票同涨同跌的天数占比
- **领先滞后关系**：判断哪只股票的走势更领先

### 资金共振
- **龙虎榜席位重叠**：分析两只股票是否被相同的游资/机构关注
- **共同席位数量**：出现在两只股票龙虎榜上的相同席位数量

## 数据源

- **股票行情**：腾讯财经 API（前复权日线数据）
- **股票名称**：东方财富缓存（TK_EastMoney/stock_names_cache.json）
- **龙虎榜数据**：TK_LHB 项目（需单独运行）

## 技术栈

- **后端**：Flask + Python 3.x
- **数据处理**：pandas, numpy, scipy
- **前端**：原生 HTML/CSS/JavaScript
- **数据源**：腾讯财经 API, 东方财富 API

## 注意事项

1. **代理设置**：使用 `session.trust_env = False` 绕过 Clash 代理
2. **缓存有效期**：股票名称缓存30天有效
3. **龙虎榜数据**：需要先运行 TK_LHB 项目获取数据
4. **分析周期**：默认90天，可调整（建议30-250天）
