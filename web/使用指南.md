## 常见问题

### Q1: 双击 start.bat 后窗口闪退

**原因**：Python 未安装或未添加到环境变量

**解决方法**：
```bash
# 检查 Python 是否安装
python --version

# 如果提示找不到命令，需要安装 Python 或添加到 PATH
```

### Q2: 提示 "ModuleNotFoundError: No module named 'flask'"

**原因**：缺少依赖包

**解决方法**：
```bash
cd C:\Users\qqqff\Downloads\CCVSCODE\TK_StockRelation
pip install -r requirements.txt
```

### Q3: 浏览器显示 "连接失败"

**原因**：后端服务未启动或端口被占用

**解决方法**：
1. 确认命令行窗口显示 "Running on http://127.0.0.1:5000"
2. 检查 5000 端口是否被占用：
   ```bash
   netstat -ano | findstr :5000
   ```
3. 如果被占用，修改 `app.py` 中的端口号：
   ```python
   app.run(host='0.0.0.0', port=5001, debug=True)  # 改为 5001
   ```

### Q4: 分析时提示 "获取数据失败"

**原因**：网络问题或 AKShare API 限流

**解决方法**：
1. 检查网络连接
2. 稍后重试
3. 检查代理设置（如果使用了代理）

## 使用说明

### 1. 输入股票代码

- **股票代码1**：输入第一只股票代码（如：300394）
- **股票代码2**：输入第二只股票代码（如：300308）
- **分析天数**：选择分析周期（10-365天，默认90天）

### 2. 查看分析结果

点击"开始分析"后，系统会显示：

- **综合评分**：0-100分，评级 A/B/C/D
- **价格联动**：相关系数、同涨同跌率、领先滞后关系
- **资金共振**：龙虎榜席位重叠情况
- **分析结论**：智能生成的投资参考建议

## 高级配置

### 修改端口

编辑 `app.py` 最后一行：
```python
app.run(host='0.0.0.0', port=5000, debug=True)  # 修改 port 参数
```

### 关闭调试模式（生产环境）

```python
app.run(host='0.0.0.0', port=5000, debug=False)
```

### 使用 Gunicorn（Linux/Mac）

```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

## 停止服务

- 在命令行窗口按 `Ctrl + C`
- 或直接关闭命令行窗口
