# 股票关联度分析工具 - 故障排查指南

## 当前问题：无法获取股票数据

### 原因分析

AKShare API 可能遇到以下问题：
1. **API 限流**：短时间内请求过多
2. **网络不稳定**：东方财富服务器响应慢
3. **数据源维护**：API 临时不可用

### 解决方案

#### 方案1：使用演示模式（推荐）

运行演示脚本，使用模拟数据查看功能：

```bash
cd TK_StockRelation
python demo.py
```

这会生成一个完整的分析报告（使用模拟数据）。

#### 方案2：稍后重试

等待 5-10 分钟后再次尝试，API 可能恢复正常。

#### 方案3：检查网络

```bash
# 测试能否访问东方财富
curl http://quote.eastmoney.com

# 如果无法访问，检查防火墙或网络设置
```

#### 方案4：使用其他数据源

修改 `data_sources/akshare_api.py`，使用备用数据源（需要手动实现）。

## Web 界面使用

即使后端 API 暂时不可用，你仍然可以：

1. **查看界面设计**
   - 双击打开 `web/index_standalone.html`
   - 查看完整的 UI 设计

2. **等待 API 恢复**
   - 保持 Web 服务运行
   - 定期尝试分析功能

## 临时解决方案

如果急需使用，可以：

1. 手动下载股票数据（CSV 格式）
2. 修改代码读取本地文件
3. 或使用 demo.py 生成的模拟报告作为参考
